## Installation

To run this portfolio locally, follow these steps:

1. **Clone the repository**:

   ```bash
   git clone https://github.com/ZarinaAttaria/Portfolio.git
   ```

2. **Execute the frontend**:
   cd frontend,
   npm install,
   npm run dev

3. **Execute the backend**:
   cd backend,
   npm install,
   npm start
